Que son los codigos html

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos

Para que nos sirven los codigos html

Los códigos html son el lenguaje universal que se utiliza para programar y elaborar sitios web. Las siglas HTML significan “Hyper Text Markup Language” o Lenguaje Marcado de Hipertexto. Este tipo de códigos genera sitios web estáticos, aunque usados con otros lenguajes de programación se pueden crear sitios dinámicos.

Codigos basicos html

  • html. este codigo indica el comienzo y finalidad de un archivo HTML
  • head. indica el empiezo y el fin de un emcabesado(generalmente titulo)
  • title. indica un titulo
  • body, indica el comienzo y fin de un cuerpo de pagina
  • p. indica comienzo y fin de un parrafo
  • br. permite saltar una linea(se llama quiebre de linea)
  • hn . tipo de subtitulo

Tipos de letras

  • B negrita
  • L cursiva
  • blink Parapadeante
  • strong Enfatizada
  • ul indica un comienzo de una lista desordenada(puntos) dentro de ellos cada item empiesa con li para empezar y terminar una lista
  • ol indica el comienzo de una lista enumerada (numeros) dentro de ellos cada item empieza con li para empezar y tereminar una lista

CSS

CSS, en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. Estos codigos nos serviran para decorar nuestras paginas o app que tengamos
